# Deepfake Protection App

## 📌 Overview
딥페이크 악용을 방지하기 위해  
이미지에 적대적 노이즈(Adversarial Noise)를 적용하여  
사전적으로 보호하는 애플리케이션입니다. 

 🏗 Architecture
- Frontend (iOS - Swift)
- Backend (Spring Boot)
- AI Server (Python - FastAPI)


📁 Project Structure

app/     # iOS App
server/  # Spring Boot Backend
model/   # AI Model Server


⚙️ Tech Stack
* Frontend: Swift
* Backend: Java, Spring Boot
* AI: Python, PyTorch
* DB: MySQL
* Infra: AWS (EC2, S3), Docker 


🚀 Features
* 이미지 업로드
* AI 기반 딥페이크 위험 분석
* 적대적 노이즈 적용
* 결과 이미지 비교 및 다운로드


🎯 Goal
딥페이크를 사전에 방지하여개인 정보 보호 및 안전한 콘텐츠 환경을 제공합니다.
